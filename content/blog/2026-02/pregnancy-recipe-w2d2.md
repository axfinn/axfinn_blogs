## 第2周 周二 菜谱

### 早餐

**主食：全麦吐司** | **饮品：酸奶** | **水果：蓝莓**

**营养价值**：
- 全麦吐司富含膳食纤维和B族维生素，GI值较低
- 酸奶含有益生菌，有助于肠道健康和消化吸收
- 蓝莓富含花青素，具有抗氧化作用，是低GI水果

**详细做法**：
1. **全麦吐司**：
   - 全麦吐司2片
   - 放入烤箱烤2-3分钟至表面微脆
   - 可搭配少量无盐黄油

2. **酸奶**：
   - 选择无糖原味酸奶200毫升
   - 可直接饮用，或搭配水果食用

3. **蓝莓**：
   - 蓝莓洗净后直接食用
   - 每次约50克

---

### 午餐（3个菜）

#### 1. 主菜：炒牛肉丝
**营养价值**：
- 牛肉富含铁和蛋白质，有助于预防孕期贫血
- 搭配胡萝卜和青椒，增加维生素和膳食纤维摄入
- 牛肉中的铁为血红素铁，吸收率较高

**详细做法**：
```
食材：牛肉200g、胡萝卜1根、青椒1个、姜2片、蒜2瓣、料酒1勺、酱油少许、淀粉少许、盐适量、食用油少许

步骤：
1. 牛肉切丝，用料酒、少许盐、淀粉腌制10分钟
2. 胡萝卜和青椒洗净，切成丝
3. 热锅下油，油热后放入姜蒜爆香
4. 放入牛肉丝，大火翻炒至变色
5. 放入胡萝卜丝和青椒丝，翻炒均匀
6. 加少许酱油调味，翻炒至熟即可
```

#### 2. 主食：米饭
**详细做法**：
```
食材：大米150g、清水225ml

步骤：
1. 大米洗净，用清水浸泡30分钟
2. 放入电饭煲，加水（1:1.5比例）
3. 选择"煮饭"模式
4. 煮好后焖5分钟再开盖
```

#### 3. 蔬菜：胡萝卜丝（凉拌）
**详细做法**：
```
食材：胡萝卜2根、蒜2瓣、醋1勺、香油少许、盐适量

步骤：
1. 胡萝卜洗净，切成细丝
2. 放入沸水中焯烫1分钟，捞出沥干
3. 蒜瓣拍碎，切成蒜泥
4. 加入蒜泥、醋、少许盐和香油
5. 拌匀即可食用
```

---

### 晚餐（2个菜）

#### 1. 汤：冬瓜肉丸汤
**营养价值**：
- 冬瓜具有利尿消肿的功效，富含维生素C和钾
- 肉丸提供优质蛋白质，易于消化吸收
- 汤品清淡爽口，有助于预防肾结石

**详细做法**：
```
食材：冬瓜300g、猪肉末150g、姜2片、葱花少许、料酒1勺、淀粉少许、盐适量、香油少许

步骤：
1. 猪肉末加入盐、料酒、淀粉搅拌上劲
2. 制成约2厘米直径的肉丸
3. 冬瓜去皮，切成块
4. 锅中加水烧开，放入冬瓜块煮沸
5. 下入肉丸，煮至肉丸浮起（约5分钟）
6. 加盐调味，出锅前滴几滴香油
7. 撒上葱花即可
```

#### 2. 蔬菜：清炒生菜
**详细做法**：
```
食材：生菜300g、蒜2瓣、盐适量、食用油少许

步骤：
1. 生菜洗净，沥干水分
2. 蒜瓣拍碎，切成蒜末
3. 热锅下油，油热后放入蒜末爆香
4. 放入生菜，大火快速翻炒
5. 炒至生菜略微变软（约30秒）
6. 加盐调味，翻炒均匀即可出锅
```

### 饮食建议

1. **酸奶**：建议选择无糖品种
2. **牛肉**：适量食用，每周2-3次为宜
3. **胡萝卜**：过量可能导致皮肤发黄，注意适量
4. **用餐时间**：
   - 早餐：7:00-8:00
   - 午餐：12:00-13:00
   - 晚餐：18:00-19:00
   - 加餐：上午10点、下午3点
5. **运动**：饭后散步有助于消化
