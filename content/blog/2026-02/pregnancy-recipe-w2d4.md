## 第2周 周四 菜谱

### 早餐

**饮品：豆浆** | **主食：粗粮饼** | **水果：香蕉**

**营养价值**：
- 自制豆浆富含植物蛋白和钙，不含添加糖
- 粗粮饼提供膳食纤维和B族维生素，有助于稳定血糖
- 香蕉富含钾元素，可补充孕期所需的电解质

**详细做法**：
1. **豆浆**：
   - 黄豆提前浸泡8小时
   - 放入豆浆机打成豆浆
   - 煮沸后小火煮5分钟

2. **粗粮饼**：
   - 准备玉米面、荞麦面等粗粮
   - 加入鸡蛋和清水调成面糊
   - 平底锅小火煎至两面金黄

3. **香蕉**：
   - 每天1根
   - 选择成熟度适中的

---

### 午餐（3个菜）

#### 1. 主菜：炖鸡腿
**营养价值**：
- 鸡腿去皮后是低脂高蛋白的肉类选择
- 富含维生素B族和矿物质
- 炖制方式清淡健康，易于消化吸收

**详细做法**：
```
食材：鸡腿2个、姜3片、葱2根、八角1个、料酒1勺、酱油2勺、盐适量

步骤：
1. 鸡腿洗净，在表面各划两刀
2. 冷水下锅，加姜片和料酒焯水
3. 煮开后撇去浮沫，捞出洗净
4. 锅中加水，放入鸡腿、姜片、葱段、八角
5. 加入酱油和料酒
6. 小火炖30-40分钟至熟烂
7. 加盐调味，大火收汁即可
```

#### 2. 主食：米饭
**详细做法**：
```
食材：大米150g、清水225ml

步骤：
1. 大米洗净，用清水浸泡30分钟
2. 放入电饭煲，加水（1:1.5比例）
3. 选择"煮饭"模式
4. 煮好后焖5分钟再开盖
```

#### 3. 蔬菜：炒青菜
**详细做法**：
```
食材：青菜300g、蒜2瓣、盐适量、食用油少许

步骤：
1. 选择小白菜或油菜
2. 洗净后沸水焯烫1分钟
3. 捞出沥干水分
4. 蒜瓣拍碎，切成蒜末
5. 热锅下油，油热后放入蒜末爆香
6. 放入青菜，大火快速翻炒
7. 加盐调味，翻炒均匀即可出锅
```

### 晚餐（2个菜）

#### 1. 汤：豆腐鱼头汤
**营养价值**：
- 鱼头富含DHA和胶原蛋白，对胎儿大脑发育有益
- 豆腐提供植物蛋白和钙，营养互补
- 汤品鲜美易消化，是孕期的营养补充佳品

**详细做法**：
```
食材：鱼头1个（约300g）、豆腐200g、姜3片、葱2根、料酒1勺、盐适量

步骤：
1. 鱼头处理干净，洗净备用
2. 热锅下油，放入姜片
3. 放入鱼头，煎至两面金黄
4. 加入料酒和足量开水（没过鱼头）
5. 大火烧开后转小火煮20分钟
6. 放入切块的豆腐，继续煮10分钟
7. 加盐调味，撒上葱花即可
```

#### 2. 蔬菜：凉拌木耳
**详细做法**：
```
食材：干木耳30g、蒜2瓣、醋1勺、香油少许、盐适量

步骤：
1. 干木耳提前用温水泡发1小时
2. 泡发后洗净，摘去根部
3. 沸水焯烫2分钟捞出
4. 放入冷水中过凉，沥干水分
5. 加入蒜泥、醋、少许盐和香油
6. 拌匀即可食用
```

### 饮食建议

1. **豆浆**：自制豆浆比外卖更健康
2. **鸡腿**：要去皮减少脂肪摄入
3. **木耳**：有活血作用，不宜过量，每次约30g即可
4. **坚果**：每天一小把补充优质脂肪
5. **用餐时间**：
   - 早餐：7:00-8:00
   - 午餐：12:00-13:00
   - 晚餐：18:00-19:00
   - 加餐：上午10点、下午3点
6. **运动**：饭后散步15-30分钟，有助于血糖控制
