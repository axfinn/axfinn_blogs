    我开发了一个超实用的开发者工具箱
    • Mar 5, 2026
    • 2 minutes read

    作为一名开发者，日常工作中有太多琐碎但必不可少的小工具：JSON 格式化、URL 编解码、正则测试、Diff 对比……每次都要打开不同的网站，甚至有些还要付费才能使用。

    于是我干脆自己动手，花了些时间做了一个开发者工具箱——现在它已经基本成型了，今天来给大家介绍一下！

    🌐 在线体验

    在线地址：https://t.jaxiu.cn

    GitHub：https://github.com/axfinn/devtools

    🎯 核心功能一览

    这个工具箱目前包含 20+ 实用工具，涵盖了开发过程中的各种场景：

    开发必备

    工具 功能
    JSON 工具 格式化、压缩、校验、转 Go Struct / TypeScript Interface、JSON Path 查询
    Diff 对比 文本对比，支持字符级、单词级、行级差异高亮
    正则测试 实时匹配测试、常用正则模板
    SSH 终端 远程 SSH 连接，支持密码/密钥认证
    Base64 文本/图片 Base64 编解码
    URL 编解码 URL Encode/Decode、参数构建
    时间戳转换 Unix 时间戳与日期时间互转

    文本处理

    工具 功能
    文本转换 大小写转换、排序、去重、行处理
    批量替换 多规则同时应用，支持正则表达式

    在线工具

    工具 功能
    Mock API 创建 Mock 端点，自定义响应状态码和响应体
    Markdown 实时预览、导出 HTML/PDF，支持创建分享链接
    Mermaid 图表 实时渲染、导出 SVG/PNG
    Excalidraw 画图 云端保存、密码保护、导出 PNG/SVG
    共享粘贴板 临时分享，支持过期时间和访问次数限制
    短链生成 自定义 ID、访问次数限制、过期时间

    生活小工具

    工具 功能
    IP/DNS 查询 查看当前 IP、域名 DNS 解析
    孕期管理 孕周计算、产检提醒
    每日菜谱 菜谱推荐、分类浏览
    生活记账 收支记录、趋势分析
    血糖监测 血糖记录、达标率统计
    二维码 生成器 + 图片中二维码识别

    社交功能

    工具 功能
    聊天室 实时聊天，密码保护，图片/视频上传

    🔒 安全与限流

    我也很重视安全问题，所以内置了这些保护机制：

    • IP 限流：基于 IP 的速率限制，防止滥用
    • 密码保护：粘贴板、聊天室、Markdown 分享、Excalidraw 都支持
    • 访问次数限制：可设置查看/点击次数上限
    • 自动清理：每小时清理过期数据，7 天前的聊天室自动删除

    🛠️ 技术栈

    • 前端：Vue 3 + Vite + Element Plus + TailwindCSS
    • 后端：Go Gin + SQLite
    • OCR：Python FastAPI + RapidOCR + OpenCV
    • 部署：Docker + Docker Compose

    🚀 快速部署

    # 克隆项目
git clone https://github.com/axfinn/devtools.git
cd devtools

# 一键启动
docker-compose up -d

# 访问
open http://localhost:8082
